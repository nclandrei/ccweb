ccweb

Bootstrap Claude Code web environments in one command.

Toolchains

Extras

Command

$ uvx ccweb init

After running the command

  1. Commit: git add scripts/ .claude/ && git commit -m "Add ccweb setup"
  2. Push to your repo
  3. Start a Claude Code web session — it auto-provisions the VM
  4. Run ccweb doctor to verify
